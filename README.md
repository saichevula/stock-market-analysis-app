# stock-market-analysis-app
# STOCK MARKET ANALYSIS APP

My second major Python project that analyzes and compares stocks using historical data.

## Features
- Closing price chart
- 50-day moving average
- Daily returns
- Annualized volatility summary

## How to Run
pip install yfinance matplotlib numpy
python stock_app.py
